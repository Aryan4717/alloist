# Alloist Landing

YC-style landing page for Alloist – the AI permission layer for the Internet.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001).

## Build (static export)

```bash
npm run build
```

Output is in `out/`. Deploy to any static host (Vercel, Netlify, GitHub Pages).
